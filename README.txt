== Description ==

It automatically set all external links to "nofollow" in website content.

Seo friendly.

Plugin Functionality: 

1) Activate it from backend and it will automatically add nofollow in external links used in content.

2) No additional settings required.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation, you can go through wordpress post/page backend.
